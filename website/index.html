About The Engine

OpenWare is a compact C++17 engine focused on clarity and expandability. It gives you a clean base for gameplay experiments, retro visuals, and tool growth.

Built with OpenGL + SDL2, it includes rendering, scene management, rigid body physics, resource loading, and in-engine debug UI.
