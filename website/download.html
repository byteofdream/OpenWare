Get The Source

Download OpenWare from GitHub:

https://github.com/byteofdream/OpenWare

Clone command:

git clone https://github.com/byteofdream/OpenWare.git

Dependencies (Fedora)

sudo dnf install gcc-c++ cmake ninja-build SDL2-devel mesa-libGL-devel pkgconf-pkg-config

Build

cmake -S . -B build_ninja -G Ninja
cmake --build build_ninja -j4

Run

./build_ninja/OpenWareEngine

Keep your game logic in src/main.cpp and refactor into your own game modules as needed.
